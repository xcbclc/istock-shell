# 全局

## ai Ai

发送ai消息

### 用法

`ai:<消息内容>`

### 参数

|   名称   | 描述               |  类型  | 是否可选 | 默认值 | 可选值 |
| :------: | :----------------- | :----: | :------: | :----: | :----: |
| 消息内容 | 发送给Ai的消息内容 | string |  false   |        |        |

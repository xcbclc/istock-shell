# 全局

## mlcz 命令查找

查找当前域下命令

### 用法

`mlcz [命令] <-mc [名称]>`

### 选项

|   选项参数    | 名称 | 描述                           |  类型  | 是否可选 | 默认值 | 可选值 |
| :-----------: | :--: | :----------------------------- | :----: | :------: | :----: | :----: |
| `-mc，--名称` | 名称 | 需要查询的命令名称，通配符匹配 | string |   true   |        |        |

### 命令示例

`mlcz lshq -mc 历史`

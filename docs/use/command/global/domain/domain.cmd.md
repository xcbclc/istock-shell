# 全局

## yycz 应用查找 
查看指定应用数据

### 用法

`yycz [-gl <正则表达式|字符串>] [-sl <数量>]`

### 参数

过滤：`-gl，--过滤`

用正则或字符串匹配过滤菜单列表

| 参数类型 | 是否可选 | 默认值 | 可选值 |
| :--: | :--: | :--: | :--: |
| string，RegExp | true |  |  |

数量：`-sl，--数量`

限制展示查看应用列表的数量

| 参数类型 | 是否可选 | 默认值 | 可选值 |
| :--: | :--: | :--: | :--: |
| number | true |  |  |
## yyjr 应用进入 
进入指定应用

### 用法

`yyjr <应用名或应用显示名称>`

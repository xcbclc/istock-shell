# 网站导航

## cjdh 财经导航 


### 用法

`cjdh [标签]`

### 参数
| 名称 | 描述 | 类型 | 是否可选 | 默认值 | 可选值 |
| :--: | :-- | :--: | :--: | :--: | :--: |
| 标签 | 财经导航标签 | string | true |  | 门户,报纸,杂志,新媒体,官方,债券,理财,期货,黄金,工具,宏观数据,市场数据,研报,选股器 |


### 命令示例

`cjdh 门户`
